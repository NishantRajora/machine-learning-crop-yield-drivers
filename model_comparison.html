// Overall Takeaway
All models achieved very high R² scores (0.997–0.999), indicating the dataset contains strong correlations between input features (rainfall, fertilizer, humidity, crop type) and yield. When the signal is this strong, even Linear Regression can perform well — the key differentiator is how each model handles residual variance and outlier predictions, which is why the RMSE gap is so large.
